# Community Feedback

PoshUI is actively developed based on user feedback and real-world use cases. If you have a feature request or a specific use case that isn't currently supported, please join the conversation:

- [Discussions](https://github.com/asolutionit/PoshUI/discussions)
- [Feature Requests](https://github.com/asolutionit/PoshUI/issues?q=is%3Aissue+label%3Aenhancement)

Your feedback helps shape the future of PoshUI.
